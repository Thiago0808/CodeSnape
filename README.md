Welcome to **CodeSnape**, where you can store snippets of your most frequently used code in your own account, giving you quick and easy access whenever you need them in your projects.

Add, edit, and delete any snippet by giving it a title and selecting its programming languages. All snippets can be viewed, edited, and filtered on the main page, along with their tags.

<h1 align="center">Usage Instructions</h1>

To use CodeSnape, you need to add the `codesnape.sql` file to your computer in order to connect to the database required for the website to work:

**Method 1:**  
Copy the code inside the `codesnape.sql` file, open your terminal, start your database using the command:
mysql -u root -p


Then paste the SQL code and execute it.

**Method 2:**  
Download the `codesnape.sql` file and import it using a database management tool such as **phpMyAdmin**.
